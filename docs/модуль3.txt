Модуль 3 генерирует нагрузки и массы и возвращает структуры данных, которые Модуль 4 превращает в текст .mct.

Часть 1. Нагрузки от пролетных строений
Есть 2 схемы, которые должны быть сгенерированы программой:
1) схема, в которой нагрузки только постоянные
2) схема, в которой нагрузки постоянные и временные

Для схемы, в которой нагрузки только постоянные
Значения постоянных нагрузок берутся из исходных данных и прикладываются к узлам, которые расположены в уровне центра тяжести пролета (если рамок две, то уровни центра тяжести могут быть разными). Массы только от постоянных нагрузок прикладываются к соответствующим отметкам (смотри исходные данные). По x, y и z могут быть разные отметки приложения нагрузок

Для схемы с постоянными и временными нагрузками
Для постоянных нагрузок процедура идентична описанной выше. Значения временных нагрузок берутся из исходных данных и прикладываются к узлам, которые расположены в уровне проезжей части (если рамок две, то уровни проезжей части могут быть разными). Массы только временных нагрузок прикладываются к соответствующим отметкам (смотри исходные данные). По x, y и z могут быть разные отметки приложения нагрузок


Часть 2. Массы от воды и разжиженного грунта

Для горизонтальных масс от разжиженного грунта и воды алгоритмы одинаковые. Отличие только в том, из каких полей берутся исходные данные.

Для вычисления масс алгоритм следующий:

1) Находятся узлы в пределах отметок *_z_top и *_z_bot (для воды или разжиженного грунта)

2) Элементы, которым принадлежат эти узлы, разделяются по типам: сваи, ростверк, тело опоры

3) Площади берутся для соответствующей зоны, к которой относится элемент и вычисляется средняя площадь элемента по линейной интерполяции.

4) Вычисляются значение массы по формуле: площадь элемента из п.3 * длину элемента * вес воды или грунта / g(ускорение свободного падения)

5) Масса делится между узлами элемента пополам и прикладывается к ним по направлениям x и y


Часть 3. Боковое давление грунта

Алгоритм для направления y и z одинаковый

1) Находятся узлы в пределах отметок y_pres_z_surf и y_pres_z_bot (или z_pres_z_surf и z_pres_z_bot) для бокового давления грунта

2) Элементы, которым принадлежат эти узлы разделяются по типам: сваи, ростверк, тело опоры

3) При помощи линейной интерполяции вычисляется средняя ширина по длине элемента. Ширины берутся для соответствующей зоны, к которой относится элемент и вычисляется средняя ширина элемента по линейной интерполяции.

4) Вычисляются значение нагрузки по формуле: Активное давление * ширину элемента. Активное давление вычисляется по формуле: γ*(TAN(45-фи/2))^2*h. h=отметка верха грунта-отметка узла
5) Так как боковое давление грунта зависит от высоты(z), то в верхнем и нижнем узле элемента будет разное его значение. Запоминаем эти значения для элементов. к которым прикладывается боковое давление (элементы из п.2)

Часть 4. Вертикальное давление грунта
1) Из полощади самого верхнего сечения ростверка вычитается площадь самого нижнего сечения стойки
2) Нагрузка вычисляется по формуле: площадь из п.1 * вес грунта * h грунта над ростверком
3) Нагрузка прикладывается в верхний узел ростверка, как вертикальная сосредоточенная сила
4) Эта же нагрузка переводится в массу по z и прикладывается к тому же узлу. Пользователю выдается предупреждение, что масса приложена только по вертикали и, если нужно приложить по горизонтали, нужно заполнить поле с разжиженным грунтом


Если соответствующие параметры *_present  имеют значение "нет", то ничего не делается.
