seismic_input.xlsx — Файл с исходными данными

mct_generator — Основной файл, единая точка входа, функции для вывода в консоль

module 1 — Считывает информацию из excel файла с исходными данными

module 2 — Строит геометрию по исходным данным, собирает 4 части в одну
module_2_part_1 — Часть 1 модуля 2 — Стержень (ростверк → стойка → ригель).
    Что делает: мешинг трёх частей вдоль оси Z, каждая со своими зонами сечений.
module_2_part_2 — Часть 2 модуля 2 — Рамки (подферменники → опорные части → вертикали → горизонтали).
    Что делает: для каждой рамки (1 и 2) строит подферменники, опорные части, вертикали и горизонтали; формирует FrameRLS.
module_2_part_3 — Часть 3 модуля 2 — забирает геометрию из файла .mct для свай и для самой опоры, перенумеровывает их
module_2_part_4 — Часть 4 модуля 2 — задает RigidLink, Constraints, Hinges

Модуль 3 прикладывает нагрузки
module_3_part_1 — Часть 1 модуля 3 — определяет координаты масс и нагрузок от пролетного строения
module_3_part_2 — Часть 2 модуля 3 — определяет массы и узлы, к которым они прикладываются, для разжиженного грунта и воды
module_3_part_3 — Часть 3 модуля 3 — определяет нагрузку от бокового давления грунта
module_3_part_4 — Часть 4 модуля 3 — определяет вертикальную нагрузку от грунта на ростверк
